# PalonaExercise

Next.js (App Router) prototype migration from the original single-file `code.html`.

## Stack

- Next.js `16.2.1`
- React `19.2.4`
- React DOM `19.2.4`
- Tailwind CSS `3.4.x`

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`

## Routes

- `/` landing page
- `/chat` chat UI prototype
- `/product` product detail prototype
